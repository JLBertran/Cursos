      Regras de identificadores de Variáveis:
       
       * Podem começar com letra, $ ou _
       * Não podem começar com numeros
       * É possível usar letras e numeros
       * É possível usar acentos e símbolos
       * Não podem conter espaços 
       * Não podem ser palavras reservadas.

     Utilize sempre nomes coerentes em suas variáveis. 
     Variáveis possuem algumas classificações sendo elas os tipos primitivos:
     Number numeros em geral
        tem 2 valores internos
          Infinity
          NaN
     String podem ser caracteres em geral que estão entre aspas ou crases
     Boolean resumindo entre true e false.
     Null
     Undefined
     Object
        Array
     Function

     utilize sempre o Typeof em caso de duvida.
